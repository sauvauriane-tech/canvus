Terminal Import Command Test

Testing the import command for the Canvus terminal endpoint.

Command Documentation

Command: import
Usage: import <filename> <target-url>
Description: Import a file to a specific Canvus page
Example: import habit.html https://canvus.app/#/file_dbmwsryv/page_main

Test Cases

Test 1: Valid Import Command

Command: import habit.html https://canvus.app/#/file_dbmwsryv/page_main

Expected: Success with import scheduled message

Test 2: Missing Arguments

Command: import habit.html

Expected: Error - missing target URL

Test 3: Invalid URL Format (Legacy)

Command: import habit.html https://canvus.app/file_dbmwsryv

Expected: Error - invalid URL format

Test 4: Valid URL Format (New)

Command: import design.png https://canvus.app/#/file_abc123/page_2

Expected: Success with import scheduled message

Usage Examples

Example 1: Import HTML file

curl -X POST https://canvus.app/terminal \
  -H "Content-Type: application/json" \
  -d '{"command": "import", "args": ["habit.html", "https://canvus.app/#/file_dbmwsryv/page_main"]}'

Example 2: Import image file

curl -X POST https://canvus.app/terminal \
  -H "Content-Type: application/json" \
  -d '{"command": "import", "args": ["design.png", "https://canvus.app/#/file_abc123/page_1"]}'

Example 3: Check help

curl -X POST https://canvus.app/terminal \
  -H "Content-Type: application/json" \
  -d '{"command": "help"}'
